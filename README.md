# 🏠 Real Estate Platform

Web application focused on lead generation for real estate, built with modern technologies.

## 🚀 Live Demo
https://antigravity-luxe-estate.vercel.app/

## 🛠️ Tech Stack
- Next.js
- TypeScript
- SCSS
- Supabase (Auth & Database)

## ✨ Features
- User registration and authentication
- Form handling with database integration
- Automated email sending after user registration
- Responsive UI

## 👨‍💻 My Role
- Developed frontend using Next.js and TypeScript  
- Implemented authentication system with Supabase  
- Integrated forms with database  
- Used AI tools (Claude Code, ChatGPT) to accelerate development and debugging  

## 🧠 What I Learned
- Authentication workflows with Supabase  
- Connecting frontend with backend services  
- Managing user data and forms
